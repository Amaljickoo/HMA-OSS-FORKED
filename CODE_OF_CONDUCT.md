# Contributor Covenant Code of Conduct

Our community is dedicated to providing an open and welcoming environment. All participants are expected to uphold the following code of conduct.

Standards
- Be respectful and considerate in language and actions.
- Be welcoming to newcomers.
- Refrain from demeaning or harassing behavior.

Reporting
If you experience or witness unacceptable behavior, please contact the repository owner via their GitHub profile: https://github.com/Amaljickoo

# HMA-OSS-FORKED

This is a forked version of HMA OSS with improvements and a contribution-friendly project layout.

## Quick links
- Repository: https://github.com/Amaljickoo/HMA-OSS-FORKED

## What this repo contains
A friendly starting point for the project with documentation, contribution guidelines, issue & PR templates, a permissive license, and a minimal CI workflow.

## How to get started
1. Clone the repository:
   ```bash
   git clone https://github.com/Amaljickoo/HMA-OSS-FORKED.git
   cd HMA-OSS-FORKED
   ```

2. Read the CONTRIBUTING.md for how to contribute.
